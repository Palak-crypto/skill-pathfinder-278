# 🚀 Skillens

Skillens is an AI-powered Resume & Job Description Analyzer that helps job seekers improve their resumes by comparing them with job descriptions and providing ATS scores, missing skills, and personalized suggestions.

## ✨ Features

- Upload Resume (PDF)
- Paste Job Description
- AI Resume Analysis
- ATS Score
- Missing Skills Detection
- Resume Improvement Suggestions
- Analysis History
- Secure Authentication

## 🛠 Tech Stack

- React 19
- TypeScript
- Vite
- TanStack Start
- Tailwind CSS
- Supabase
- OpenAI API

## 🚀 Installation

```bash
git clone https://github.com/Palak-crypto/skill-pathfinder-278.git
cd skill-pathfinder-278
npm install
npm run dev
```
